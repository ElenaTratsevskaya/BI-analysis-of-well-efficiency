### Задание конкурса<br>
**1. Анализ фонда:**</br>
изучите динамику эксплуатации скважин по месторождениям и категориям,</br>
выявите закономерности простоев и факторы стабильности добычи.</br>
**2. Оценка подрядчиков:**</br>
Оцените эффективность буровых подрядчиков через корреляцию их рейтинга с ключевыми показателями работы скважин, включая среднее время бурения,</br> 
частоту последующих ремонтов и стабильность дебитов нефти и газа, чтобы выявить подрядчиков, обеспечивающих наибольшую надёжность скважин.</br>
**3. Оптимизация ремонтов:**</br>
Исследуйте распределение причин ремонтов по типам и ответственным подразделениям, определите приоритетные причины остановок скважин.</br></br>

### Требования </br>
• Создать аналитическое приложение, состоящее минимум из 3 4 дашбордов. </br>
• Добавить заданные датасеты в модель данных приложения. </br>
• В каждом задании проанализовать данные, используя различные визуализации, представить данные в интуитивно понятном виде. </br>
• На каждом дашборде добавьте аналитическую записку с выводами, например, в визуализации «Текст». </br>

### Критерии оценки выполнения задания
**Выполнение задания**
1. Произвести анализ данных согласно заданию и подготовить аналитическую записку на дашборде с вашими выводами или рекомендациями.
2. Выполнить все 3 задания

**Техника**
1. Разнообразие и подбор типов визуализаций;
2. Использование языка P IX Meta и переменных;
3. Всестороннее использование функционала визуализаций.
4. Использование JS визуализаций.

**Дизайн**
1. Аккуратное расположение визуализаций;
2. Применение темы оформления, шрифтов, фона, палитры сочетаемых цветов и т.д.
3. Использование логичных группировок объектов и понятной навигации.

### Цвето-графическое решение</br>
Для навигации по листам были отобраны фотографии из открытого доступа и переработаны с помощью AI-инструментов.</br>
Подбор и переработка осуществлялась в палитре картин Ван Гога, например, "Звёздная ночь" и "Ночная терраса кафе", хотя в подобных ультрамариново-синих тонах с присутствием контрастного жёлто-оранжевого есть ещё полотна.</br></br>

При создании дашборда использован метод **user journey**</br></br>

Аномалии выделены оранжевым</br>
Максимальные показатели – тёмно-синим/синим</br>
Серый – основной нейтральный цвет в показателях.</br></br>


### Композиционное решение</br>

Левый блок на всех листах содержит:</br>
переключение навигации по листам с помощью изображения;</br>
набор фильтров</br></br>

Рабочая область горизонтально разделена на зоны:</br>
линия карточек к агрегированными показателями</br>
три линии детализаций в виджетах, в каждой линии по два виджета</br></br>


Блок противоположный по диагонали блоку изображений с навигацией - Аналитическая записка к каждому листу.</br></br>

Листы дашборда выполнены в соответсвии с заданием.</br></br>
